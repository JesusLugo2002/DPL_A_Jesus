## Comandos utilizados

1. Clonado del repositorio remoto
```git
git clone https://github.com/JesusLugo2002/DPL_A_Jesus.
```

2. Commit inicial:
```
git add .
git commit -m "Commit inicial"
git push
```

3. Creación de tag:
```
git tag v0.1
```
